<h1> Technology Installation </h1>
<h2>Install NodeJs: </h2> 
<h3>Windows:</h3>
Download link(LTS version) - https://nodejs.org/en/download/ 
<h3> Linux: </h3>
$ curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash - <br>
$ sudo apt-get install -y nodejs

<h1> npm run(react) </h1>
$ npm start
